# eSPA32
esp32 IoT spa controller via blynk app

## Demo video

[Video demo](https://youtu.be/)

## Installation

### Functions

* Hottub temprature control
* Individual pump control
* Lightign control
* Control everything via smartphone

### Hardware

* ESP32
* WS2812b
* Push buttons
* I2C oled display
* LEDs
* 5v powersupply
